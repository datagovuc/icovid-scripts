# ICOVID: my touch

A continuación se detalla cómo correr el flujo que genera los archivos que se cargan en Tableau.

1. Actualizar el repo local:
    ```bash
    git pull
    ```
2. En el mismo path donde se haya clonado este repo, ejecutamos lo siguiente:

    ```bash
    bash icovid.sh
    ```
